# Assets Directory

## App Icon

Place your custom app icon here as one of:
- `icon.png` (PNG format, recommended size: 64x64 or 128x128 pixels)
- `icon.icns` (macOS native format, preferred for bundled apps)

The app will automatically load the icon if it exists. If no icon is found, the app will fall back to using emoji (⚔️).

### Icon Guidelines
- Use a transparent background for best results
- Keep the design simple and recognizable at small sizes
- The icon will appear in the macOS menu bar (status bar)
